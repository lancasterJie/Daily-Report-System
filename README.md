<a name="背景介绍">
## 背景介绍
大多数组织一开始都是拥有几个核心成员的小型团队，本项目针对小型团队开发了一款可以监督并督促成员工作的邮件收发系统。目前在教学开发中应用。

<a name="项目介绍">
## 项目介绍

<a name="概要">
### 概要
定时提醒、接收与解析邮件，且能够按照预先设定好的分组，分组生成每个小组的日报文件，并可以定时等间隔地分组发送，将各组的日报发送给不同的导师。
